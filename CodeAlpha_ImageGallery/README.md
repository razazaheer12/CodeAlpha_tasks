This project is a responsive web-based image gallery showcasing a collection of BMW car images. Built with HTML, CSS, and JavaScript, it features a grid layout for displaying thumbnails, an interactive lightbox for full-size viewing with navigation controls, and a toggle button to apply a grayscale filter to all images. The design incorporates modern fonts and responsive breakpoints for optimal viewing across devices, providing a clean and engaging user experience for browsing automotive photography.

- Responsive Grid Layout: Displays images in a flexible grid that adapts to different screen sizes, ensuring optimal viewing on desktop and mobile devices.

- Grayscale Toggle: A button that applies or removes a grayscale filter to all images in the gallery for a stylistic effect.
